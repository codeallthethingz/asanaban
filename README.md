# asanaban
Kanban on Asana

## Requirements
- Node.
- Vercel - `npm -g i vercel`
- Vercel account - signup at vercel.com
- Asana account - signup at asana.com

## Running locally

You have to associated with a vercel project

`vercel`

The run dev mode

`vercel dev`

## Deployment

run `./deployProduction`
